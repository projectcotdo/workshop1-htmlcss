Lập trình Web

HTML

HTML viết tắt cho HyperText Markup Language,là một ngôn ngữ đánh dấu được thiết kế ra để tạo nên các trang web với các mẩu thông tin được trình bày trên World Wide Web.

CSS

CSS là từ viết tắt của Cascade Style Sheet, là ngôn ngữ giúp trình duyệt hiểu được các thiết lập định dạng và bố cục cho trang web.

Trang web tĩnh

Web tĩnh là những website không có hệ thống quản lý nội dung và người dùng không thể chỉnh sửa được hoặc có thể chỉnh sửa, thay đổi dữ liệu được. Tuy không được sử dụng rộng rãi nhưng khả năng tạo được 1 trang web tĩnh là căn bản đối với những ai muốn lập trình web
